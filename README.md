# Python Voice Assistant

A beginner-friendly Python Voice Assistant that listens to your voice commands and responds with speech. This project demonstrates how to use speech recognition and text-to-speech in Python, with simple commands and robust error handling.

## What is this project about?
This project is a simple voice assistant built with Python. It can listen to your voice, recognize specific commands, and respond using speech. It is designed for beginners to learn about voice input and output in Python.

## Features
- Listens for your voice commands
- Responds with speech using text-to-speech
- Supports the following commands:
  - "hello"
  - "what time is it"
  - "open google"
  - "exit"
- Handles microphone errors gracefully
- Clean, well-commented code for easy learning

## Project Files
- `main.py` — The main Python script for the voice assistant
- `requirements.txt` — List of all required dependencies
- `README.md` — Project documentation (this file)

## How to Fork
1. Click the **Fork** button at the top right of this repository on GitHub.
2. Clone your forked repository to your local machine:
  ```bash
  git clone https://github.com/your-username/Python-Voice-Assistant.git
  ```
3. Navigate to the project directory:
  ```bash
  cd Python-Voice-Assistant
  ```

## Installation Steps
1. Make sure you have Python 3 installed.
2. Install the required dependencies:
  ```bash
  pip install -r requirements.txt
  ```

## How to Run
```bash
python main.py
```

## Technologies Used
- Python 3
- [SpeechRecognition](https://pypi.org/project/SpeechRecognition/) (for voice input)
- [PyAudio](https://pypi.org/project/PyAudio/) (for microphone access)
- [pyttsx3](https://pypi.org/project/pyttsx3/) (for text-to-speech)
